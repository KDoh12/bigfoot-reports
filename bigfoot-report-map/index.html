Beware of Bigfoot

How close have bigfoot been sighted near you? Use this map to find out!

Click/tap anywhere on the map to see reports of bigfoot sightings within a 25 mile radius. You can also use the geolocate button below the zoom controls to search from your current location.

Buffer is being generated via Turf.js

  • Bigfoot data is being pulled dynamically from the ArcGIS REST API from ESRI Training Services . This data is to be used for training, demonstration, and educational purposes only.
  • State boundaries obtained from Natural Earth and transformed into an optimized GeoJSON.
